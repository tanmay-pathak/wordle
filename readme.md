# Wordle with Friends

Wordle with Friends is a fun Wordle clone built for playing with friends. It features individual play, leaderboards, and more! This app is built using Next.js, Convex, and Clerk, and it's deployed on Vercel.

## Features

- **Daily Word Puzzles**: Solve a new word puzzle every day.
- **Individual Play**: Compete with your colleagues and see who solves the puzzle the fastest.
- **Leaderboards**: Track your progress and see where you stand against others.
- **User Authentication**: Secure authentication powered by Clerk.

## Tech Stack

- **Next.js**: React framework for building the user interface.
- **Convex**: Backend for handling game logic and state management.
- **Clerk**: User authentication and management.
- **Vercel**: Hosting and deployment.

## Getting Started

To get started with Wordle with Friends, follow these steps:

### Prerequisites

- Node.js v14.x or higher
- npm v6.x or higher
- A Vercel account
- A Clerk account
- A Convex account

### Deployment

Wordle with Friends is deployed on Vercel. You can deploy your own instance by following these steps:

1. **Deploy to Vercel**:
   - Connect your GitHub repository to Vercel.
   - Add the environment variables to your Vercel project settings.
   - Deploy your app.
